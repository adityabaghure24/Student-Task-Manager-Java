# Student Task Manager (Java)

## Description
A simple Java console-based application to manage student tasks like assignments, exams, and deadlines.

## Features
- Add task
- View tasks
- Delete task
- Mark task as completed
- File saving support

## Tech Stack
- Java
- ArrayList
- File Handling

## How to Run
1. Compile the code
2. Run TaskManager.java

## Future Scope
- GUI version
